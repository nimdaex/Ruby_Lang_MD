# ภาษา Ruby

บทความนี้เป็น **คู่มือการเรียนเขียนโปรแกรมภาษา Ruby** ที่มุ่งเน้นการปูพื้นฐานผ่านการพัฒนา **Console application** เพื่อให้ผู้เรียนเข้าใจโครงสร้างหลักได้อย่างถ่องแท้ เนื้อหาครอบคลุมตั้งแต่ **ประวัติความเป็นมา** ของภาษาที่พัฒนาโดย Yukihiro Matsumoto ไปจนถึงคุณสมบัติเด่นอย่างการเป็น **ภาษาเชิงวัตถุ (OOP)** ที่มีความยืดหยุ่นสูง 

บทเรียนชุดนี้ประกอบด้วยหัวข้อสำคัญกว่า 25 หัวข้อ เช่น **การใช้ตัวแปร คำสั่งควบคุม และการจัดการหน่วยความจำอัตโนมัติ** ซึ่งเป็นรากฐานสำคัญในการต่อยอดไปสู่การเขียนโปรแกรมระดับสูงหรือการพัฒนาเว็บแอปพลิเคชัน นอกจากนี้ผู้สอนยังให้ความสำคัญกับการใช้ **เวอร์ชันล่าสุด** เพื่อให้สอดคล้องกับมาตรฐานการทำงานในปัจจุบัน โดยเน้นรูปแบบการนำเสนอที่ **เข้าใจง่ายและไม่มีค่าใช้จ่าย** สำหรับผู้ที่เริ่มต้นศึกษาด้วยตนเอง

---

## บทนำและความเป็นมาของภาษา Ruby

**ภาษา Ruby** เป็นภาษาโปรแกรมคอมพิวเตอร์ระดับสูงที่มีลักษณะเป็น **ภาษาแบบไดนามิก (Dynamic Programming Language)** และมีคุณสมบัติในการ **ปรับแต่งโครงสร้างของตนเองได้ในขณะที่โปรแกรมกำลังทำงาน (Self-modifying)** ภาษา Ruby ถูกออกแบบและพัฒนาขึ้นในช่วงกลางทศวรรษที่ 1990 โดย **Yukihiro Matsumoto** ณ ประเทศญี่ปุ่น โดยมีจุดประสงค์เพื่อเป็นภาษาสำหรับการเขียนโปรแกรมเพื่อวัตถุประสงค์ทั่วไป (General-purpose programming)

ในเชิงปรัชญาของการออกแบบ ภาษานี้ได้รับอิทธิพลอย่างโดดเด่นจากภาษาคอมพิวเตอร์หลากหลายประการ อาทิ ภาษา Perl, Smalltalk, Eiffel, Ada และ Lisp ส่งผลให้ Ruby เป็นภาษาที่สนับสนุนการเขียนโปรแกรมในหลากหลายรูปแบบ (Multi-paradigm) ซึ่งครอบคลุมทั้ง:

* **การเขียนโปรแกรมเชิงวัตถุ (Object-oriented Programming):** ที่มีความเป็นออบเจ็คมาตั้งแต่กำเนิด
* **การเขียนโปรแกรมแบบฟังก์ชัน (Functional Programming)**
* **การเขียนโปรแกรมแบบสั่งการ (Imperative Programming)** นอกจากนี้ ตัวภาษายังมีระบบการจัดการหน่วยความจำแบบอัตโนมัติ (Automatic memory management/Garbage collection) และมีประเภทข้อมูลแบบไดนามิกที่ยืดหยุ่นต่อการใช้งาน

## คุณลักษณะทางเทคนิคและสถาปัตยกรรมของภาษา

สำหรับการศึกษานี้ จะอ้างอิงเนื้อหาตามมาตรฐานของภาษา Ruby เวอร์ชัน **Stable release (เวอร์ชัน 3.4.8)** ซึ่งเป็นเวอร์ชันที่ได้รับการสนับสนุนอย่างเป็นทางการและมีคุณสมบัติที่ครบถ้วนสำหรับการพัฒนาระบบในระดับสากล โดยมีคุณลักษณะเด่นทางโครงสร้างและกลไกการทำงานที่สำคัญ ดังนี้:

* **แนวคิดเชิงวัตถุและโครงสร้างทางภาษา:** ครอบคลุมตั้งแต่โครงสร้างพื้นฐาน ตัวแปร (Variables) และขอบเขตของตัวแปร (Scope) ไปจนถึงการนิยามเมธอด (Methods)
* **การสืบทอดและโมดูล:** สนับสนุนการสืบทอดคลาส (Class inheritance) และการใช้ **Mixins** เพื่อเพิ่มขีดความสามารถของคลาสโดยไม่ต้องพึ่งพาการสืบทอดแบบหลายทาง (Multiple inheritance)
* **กลไกการประมวลผลระดับสูง:** ประกอบด้วยการใช้ Iterators สำหรับการวนซ้ำ, Closures สำหรับการจัดการฟังก์ชัน และระบบ Exception handling เพื่อการควบคุมข้อผิดพลาดของโปรแกรมอย่างมีประสิทธิภาพ
* **การจัดการข้อมูลและหน่วยความจำ:** มีระบบการจัดเก็บขยะ (Garbage collection) และการทำงานกับโครงสร้างข้อมูลที่ซับซ้อน เช่น อาเรย์ (Array) และแฮช (Hash)

## เป้าหมายของการเรียนรู้และการนำไปต่อยอด

เนื้อหาในหลักสูตรนี้มุ่งเน้นการพัฒนาในรูปแบบ **Console application** ซึ่งเป็นการรับค่าและแสดงผลในรูปแบบข้อความ (Text-based) เป็นหลัก แม้จะเป็นการแสดงผลที่เรียบง่าย แต่มีวัตถุประสงค์หลักเพื่อให้ผู้เรียนเกิดความเข้าใจอย่างลึกซึ้งใน **หลักการพื้นฐาน (Fundamentals)** และโครงสร้างภายในของภาษา 

การสร้างรากฐานที่แข็งแกร่งในระดับคอนโซลนี้ ถือเป็นปัจจัยสำคัญที่จะช่วยให้โปรแกรมเมอร์สามารถนำไปประยุกต์ใช้ในการสร้างแอปพลิเคชันที่มีความซับซ้อนสูงขึ้นในอนาคต ไม่ว่าจะเป็นการพัฒนาส่วนต่อประสานกับผู้ใช้แบบกราฟิก (**GUI - Graphical User Interface**) หรือการพัฒนา **เว็บแอปพลิเคชัน (Web Applications)** ได้อย่างมีประสิทธิภาพ

### เปรียบเทียบกลไกการทำงาน:

> การเรียนรู้ภาษา Ruby เปรียบเสมือน **การเรียนรู้วิชาสถาปัตยกรรมที่ผู้ออกแบบไม่ได้เพียงแค่สร้างอาคาร แต่ยังมอบชุดเครื่องมือที่สามารถปรับเปลี่ยนรูปทรงของอาคารได้ตามความต้องการในขณะที่มีผู้คนอาศัยอยู่ (Dynamic/Self-modifying)** โดยเริ่มจากการวางรากฐานโครงสร้างที่แข็งแรงในชั้นใต้ดิน (Console) ก่อนที่จะต่อยอดขึ้นไปเป็นอาคารที่สวยงามและซับซ้อน (GUI/Web) ในภายหลัง
